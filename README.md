Linear Assignement Problem tracker
==================================

Python implementation of the particle tracking alogorithm described in
[Jaqaman et al. Nature Methods (2008)](http://www.nature.com/nmeth/journal/v5/n8/full/nmeth.1237.html).

Dependence
----------

- numpy
- scipy
- scikit-image
- pandas
- scikit-learn
